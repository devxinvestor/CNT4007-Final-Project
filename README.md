-- Project Summary
We all created a File Sharing Systemn that allowed use to connect to each other on a network and recieve and send files to each other. We use tcp connections and we can run it my running the perrprocess file within the main terminal. When connected we are able to send request, have, not interested, interested, and piece messages. We can also recieve have messages and update bitfields that are necessary.





-- Running Project
1. The group cds in the main directory
2. they run this command python3 peerProcess.py (peerID)
